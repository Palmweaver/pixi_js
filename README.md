# PixiJs

A gem for bundling the pixi.js library into Rails projects

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'pixi_js'
```

Add this line to your application.js file

    //= require pixi

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pixi_js
